Для работы чат-бота вам понабится файл с окружением venv. 
На гитхабе его нет из-за размера. Вам нужно один раз создать этот файл и дальше сможете спокойно работать с чат-ботом.
(папку с окружением нужно называть именно venv, т.к. это название прописано в gitignore)

1)  Скачиваете репозиторий с гитхаба (Clone Git Repository)
2)  Создаёте отдельную папку(не в клоне репозитория), открываете её через вс код, вызываете терминал ctrl+ё (если он у вас не cmd смотрите первый гайд)

Вводите команды:
python -m venv venv
.\venv\Scripts\activate.bat
pip install rasa
rasa init --no-prompt
pip install pymorphy2
3)  Когда всё установилось копируете из второй папки папку venv и втавляете её в первую папку (клон репозитория)
4)  Вы можете приступать к работе, команды для работы:

1_терминал
запускаем окружение: .\venv\Scripts\activate.bat
запускаем сервер с кастомными actions: rasa run actions

2_терминал
запускаем окружение: .\venv\Scripts\activate.bat
обучаем модель, если нужно: rasa train --fixed-model-name NAME  
запускаем обученную модель: rasa shell -m models/NAME.tar.gz --endpoints endpoints.yml
запускаем обученную модель с детальной инормацией по интентам и сущностям: rasa shell -m models/NAME.tar.gz --endpoints endpoints.yml nlu
